# Security & Take-down Policy / 安全与版权申诉策略

## Reporting a Take-down or Academic Issue / 报告学术机密或版权问题

This repository compiles review materials for Northwestern Polytechnical University (NWPU) Software School. 

If you identify any content that violates copyrights (e.g., proprietary slides, textbook printouts) or violates academic integrity codes (e.g., active private exam sheets or homework keys), please do **not** open a public issue.

Please report issues privately by emailing the maintainers or using GitHub's private reporting feature so we can remove the content immediately.

本仓库汇集了西北工业大学软件学院的复习突击资料。

如果您在仓库中发现任何侵犯版权（如受版权保护的幻灯片、教材扫描件）或违反学术诚信规范（如未公开的考卷、作业答案）的内容，请勿通过公开 Issue 报告，请通过电子邮件联系维护者或使用 GitHub 的私有报告功能，我们将立即核实并删除相关内容。
