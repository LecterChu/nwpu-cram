一、路由 & 网络层
----------

### BGP

边界网关协议，用于不同自治系统之间交换路由信息。  
它不基于最短路径，而是基于策略选择路由。

### OSPF

开放最短路径优先协议，用于自治系统内部路由。  
基于链路状态算法，收敛快、无环路。

### RIP

路由信息协议，用于小型网络。  
基于距离向量算法，以跳数为度量，最大跳数为 15。

### CIDR

无类域间路由，取消 IP 地址分类，使用可变长前缀。  
可以提高地址利用率，并通过路由聚合减少路由表条目。

### VLSM

可变长子网掩码，允许在一个主网络中使用不同长度的子网掩码。  
目的是减少 IP 地址浪费。

### NAT

网络地址转换，将私有 IP 映射为公网 IP。  
使多个主机共享一个公网 IP 访问互联网。

### ICMP

互联网控制消息协议，用于网络错误报告和诊断。  
典型应用是 ping 命令（Echo 请求 / 应答）。

二、传输层 & 可靠传输
------------

### TCP

传输控制协议，面向连接、可靠的传输层协议。  
提供确认、重传、流量控制和拥塞控制机制。

### UDP

用户数据报协议，无连接、不可靠的传输层协议。  
不提供确认和重传，适合实时通信。

### RTT

往返时间，指数据从发送到收到确认的时间间隔。  
TCP 使用 RTT 计算超时重传时间（RTO）。

### TTL

生存时间，IP 包经过一个路由器减 1。  
当 TTL 为 0 时，路由器丢弃该包，防止环路。

### MTU

最大传输单元，数据链路层一帧能承载的最大数据量。  
典型以太网 MTU 为 1500 字节。

### MSS

最大段大小，TCP 段中数据部分的最大字节数。  
MSS = MTU − IP 头 − TCP 头，以太网中通常为 1460。

### cwnd

拥塞窗口，发送方根据网络拥塞程度动态调整。  
用于 TCP 拥塞控制（慢启动、拥塞避免）。

### rwnd

接收窗口，接收方通知发送方的剩余缓存大小。  
用于 TCP 流量控制，防止接收方被淹没。

### rdt

可靠数据传输协议，是理论模型。  
解决分组损坏、丢失、乱序等问题，为 TCP 提供基础。

三、链路层 & 无线
----------

### MAC 地址

物理地址，标识网络设备的唯一地址。  
工作在数据链路层，共 48 位。

### CSMA/CD

载波监听多点接入 / 碰撞检测，用于有线以太网。  
先听后说，边发边听，碰撞后随机重发。

### CSMA/CA

载波监听多点接入 / 碰撞避免，用于无线局域网。  
先听后说，随机退避，使用 ACK 确认。

### FCS

帧检验序列，用于差错检测。  
发送方计算 CRC 值填入 FCS，接收方重新计算并比对。

### SOH

首部开始，表示帧或帧首部的起始位置。  
属于数据链路层的控制字符。

### EOT

传输结束，表示一帧的结束。  
接收方收到 EOT 认为帧传输完成。

### ESC

转义字符，用于字节填充。  
当数据中出现控制字符时，在其前面插入 ESC。

四、网络管理 & 辅助协议
-------------

### DNS

域名系统，将域名解析为 IP 地址。  
采用分布式、层次化结构。

### DHCP

动态主机配置协议，自动为主机分配 IP 地址。  
工作流程：发现 → 提供 → 选择 → 确认。

### SNMP

简单网络管理协议，用于网络设备管理。  
包含 MIB、SMI、协议本身和安全管理四部分。

### ARP

地址解析协议，根据 IP 地址获取 MAC 地址。  
工作在数据链路层和网络层之间。

五、应用层协议
-------

### HTTP

超文本传输协议，用于 Web 数据传输。  
基于 TCP，采用请求/响应模式。

### FTP

文件传输协议，用于文件上传和下载。  
使用两个连接：控制连接（21）和数据连接（20）。

### SMTP

简单邮件传输协议，用于发送邮件。  
基于 TCP，采用推（push）方式。

### POP3

邮局协议第 3 版，用于将邮件从服务器下载到本地。  
下载后邮件通常从服务器删除。

### IMAP

互联网消息访问协议，用于在服务器端管理邮件。  
支持多设备同步，邮件保留在服务器上。

### BitTorrent

比特洪流协议，一种 P2P 文件共享协议。  
将文件分块，从多个对等方同时下载。

一、路由 & 网络层基础
============

1️⃣ BGP
-------

**问题：请简述 BGP 是什么，以及它的主要作用。**

**参考答案**  
BGP（边界网关协议）是 **AS（自治系统）之间的外部网关协议**。

* 作用：在不同 AS 之间交换路由可达性信息

* 特点：**策略驱动**（不是最短路径优先）

* 典型场景：运营商之间、跨国互联

* * *

2️⃣ OSPF
--------

**问题：OSPF 属于哪类协议？简述其特点。**

**参考答案**  
OSPF（开放最短路径优先）是 **内部网关协议（IGP）**。

* 算法：**链路状态（LS）**

* 特点：
  
  * 收敛快
  
  * 无路由环路
  
  * 支持分层（区域）
  
  * 基于带宽计算开销

* * *

3️⃣ RIP
-------

**问题：RIP 的最大缺点是什么？为什么？**

**参考答案**  
RIP（路由信息协议）基于 **距离向量（DV）算法**。  
主要缺点：

* **最大跳数限制 15**（不适合大型网络）

* **收敛慢**、存在 **计数到无穷** 问题

* 使用跳数作为唯一度量（不科学）

* * *

4️⃣ CIDR 与 VLSM
---------------

**问题：CIDR 和 VLSM 分别解决了什么问题？**

**参考答案**

* **VLSM**：允许一个主网络内使用 **不同长度子网掩码**，提高 IP 利用率

* **CIDR**：**聚合路由**，把多个连续子网聚合成一条路由，减少路由表大小

> 一句话区分：VLSM 对内精细划分，CIDR 对外聚合通告。

* * *

5️⃣ NAT
-------

**问题：NAT 的作用是什么？它有什么优缺点？**

**参考答案**  
NAT（网络地址转换）将私有 IP 映射为公网 IP。

* 优点：
  
  * 缓解 IPv4 地址短缺
  
  * 隐藏内网结构

* 缺点：
  
  * 破坏端到端通信
  
  * 某些协议（如 FTP、IPsec）穿透困难

二、传输层 & 可靠传输
============

6️⃣ TCP / UDP
-------------

**问题：简述 TCP 与 UDP 的 3 个主要区别。**

**参考答案**

| TCP        | UDP      |
| ---------- | -------- |
| 面向连接       | 无连接      |
| 可靠、有序      | 不可靠、可能乱序 |
| 有流量 / 拥塞控制 | 无        |
| 慢、开销大      | 快、轻量     |

> 举例：TCP → HTTP / FTP；UDP → DNS / 视频流

* * *

7️⃣ RTT
-------

**问题：RTT 是什么？它在 TCP 中起什么作用？**

**参考答案**  
RTT（往返时间）是从发送数据到收到 ACK 的时间差。  
TCP 用它：

* 计算 **超时重传时间（RTO）**

* 判断网络延迟

* 影响拥塞窗口增长

* * *

8️⃣ cwnd / rwnd / swnd
----------------------

**问题：cwnd、rwnd、swnd 分别代表什么？它们之间的关系是什么？**

**参考答案**

* **cwnd**：拥塞窗口（发送方根据网络拥塞程度控制）

* **rwnd**：接收窗口（接收方剩余缓存）

* **swnd**：发送窗口 = **min(cwnd, rwnd)**

> 实际能发送的数据量由 swnd 决定。

* * *

9️⃣ MSS & MTU
-------------

**问题：MSS 和 MTU 之间的关系是什么？**

**参考答案**

* **MTU**：数据链路层能携带的最大字节数

* **MSS**：TCP 段中**数据部分**的最大字节数

关系（典型以太网）：

text

MSS = MTU − IP头 − TCP头
    = 1500 − 20 − 20 = 1460

* * *

🔟 rdt
------

**问题：什么是 rdt？它解决了哪些问题？**

**参考答案**  
rdt（可靠数据传输协议）是**理论模型**，解决：

* 分组损坏 → ACK / NAK

* ACK/NAK 损坏 → 序号

* 分组丢失 → 超时重传

> rdt → GBN → SR → TCP 的基础思想

三、链路层 & 无线
==========

1️⃣1️⃣ CSMA/CD vs CSMA/CA
-------------------------

**问题：CSMA/CD 与 CSMA/CA 的区别？为什么无线用 CA，以太网用 CD？**

**参考答案**

| 协议      | 策略       | 适用场景     |
| ------- | -------- | -------- |
| CSMA/CD | **碰撞检测** | 有线以太网    |
| CSMA/CA | **碰撞避免** | 无线 Wi-Fi |

无线不能用 CD 的原因：

* 无法同时收发

* 信号衰减大

* 存在隐藏终端问题

* * *

1️⃣2️⃣ MAC 地址
-------------

**问题：MAC 地址的作用？与 IP 地址有何区别？**

**参考答案**

* **MAC 地址**：数据链路层、48 位、全球唯一、出厂固化

* **IP 地址**：网络层、逻辑地址、可变

> 同一子网内用 MAC 通信，跨子网用 IP 寻址。

* * *

1️⃣3️⃣ FCS
----------

**问题：FCS 的作用是什么？**

**参考答案**  
FCS（帧检验序列）用于 **差错检测**。

* 发送方计算 CRC

* 接收方重新计算并对比

* 不一致则**丢弃帧**（不纠错）

* * *

1️⃣4️⃣ SOH / EOT / ESC
----------------------

**问题：SOH、EOT、ESC 在数据链路层中的作用分别是什么？**

**参考答案**

* **SOH**：帧起始标志

* **EOT**：帧结束标志

* **ESC**：转义字符（防止数据与控制标志冲突）

> 典型场景：PPP / HDLC 等面向字节的协议

四、网络管理 & 辅助协议
=============

1️⃣5️⃣ ICMP
-----------

**问题：ICMP 的主要用途？举例说明。**

**参考答案**  
ICMP（互联网控制消息协议）用于 **错误报告 & 网络诊断**。  
典型例子：

* `ping`（Echo 请求 / 应答）

* `traceroute`（利用 TTL 超时）

* * *

1️⃣6️⃣ SNMP
-----------

**问题：SNMP 的 4 个核心组成部分是什么？**

**参考答案**

1. **MIB**：管理信息库

2. **SMI**：数据定义语言

3. **SNMP 协议**：通信机制

4. **安全管理**（SNMPv3）

> 常用于网络设备（路由器 / 交换机）监控。

* * *

1️⃣7️⃣ DNS
----------

**问题：DNS 的作用是什么？**

**参考答案**  
DNS（域名系统）将**域名 → IP 地址**。

* 分布式、层次化

* 典型解析：`www.baidu.com` → `39.156.66.14`

* * *

1️⃣8️⃣ DHCP
-----------

**问题：DHCP 的作用是什么？简述工作过程。**

**参考答案**  
DHCP 自动分配 IP 地址（避免手动配置）。  
工作流程（四步）：  
**发现 → 提供 → 选择 → 确认**

> 典型租期：24 小时

五、应用层协议（常考）
===========

1️⃣9️⃣ HTTP
-----------

**问题：HTTP 的特点？与 SMTP 的主要区别？**

**参考答案**

* HTTP：**拉（pull）**、每个对象独立响应

* SMTP：**推（push）**、多对象一个报文

> 考试常和 SMTP 对比。

* * *

2️⃣0️⃣ SMTP / POP3 / IMAP
-------------------------

**问题：SMTP、POP3、IMAP 分别用于什么场景？**

**参考答案**

* **SMTP**：发邮件

* **POP3**：下载邮件到本地（删除服务器副本）

* **IMAP**：服务器端同步（多设备）
